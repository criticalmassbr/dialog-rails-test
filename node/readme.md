## 🔨 instructions

to get started with the app, clone the repo:
```
$ git@github.com:firmoholanda/dialog-node.git
```

navigate into the newly created folder:
```
$ cd dialog-node
```

install dependencies:
```
$ npm i
```

start server:
```
$ node app.js
```

visit this link in your browser:
```
$ http://localhost:3001/
```
